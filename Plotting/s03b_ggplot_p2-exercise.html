Exercise 2: Line for each Country

Fix this plot so that it shows life expectancy over time with a separate line for each country.

Notice that we tried to use group_by(). We will cover that next week. But also notice that ggplot2 ignores the grouping of a tibble!

Code Provided:

gapminder %>% 
  group_by(country) %>% 
  ggplot(aes(year, lifeExp)) +
  geom_line()

Code Fixed:

gapminder %>% 
  ggplot(aes(year, lifeExp, group = country, colour = country == "Rwanda")) +
  geom_line(alpha = 1/3) +
  scale_colour_discrete("", labels = c("Other", "Rwanda"))

Exercise 3: More gdpPercap vs lifeExp

3(a) Facets

Original Code

ggplot(gapminder, aes(gdpPercap, lifeExp)) +
  geom_point(alpha = 0.2) +
  scale_x_log10()

Fixed Code

ggplot(gapminder, aes(gdpPercap, lifeExp)) +
  geom_point(alpha = 0.2) +
  scale_x_log10(labels = scales::comma_format()) + 
  facet_wrap(~ continent, scales = "free_y")

3(b) Bubble Plot

Origional Code

gapminder %>% 
  filter(continent != "Oceania") %>% 
  ggplot(aes(gdpPercap, lifeExp)) +
  facet_wrap(~ continent) +
  geom_point(alpha = 0.2) +
  scale_x_log10(labels = scales::comma_format())

Fixed Code

gapminder %>% 
  filter(continent != "Oceania") %>% 
  ggplot(aes(gdpPercap, lifeExp, size = pop, fill = continent)) +
  facet_wrap(~ continent, nrow = 1) +
  geom_point(alpha = 0.2, shape = 21) +
  scale_x_log10(labels = scales::comma_format()) +
  scale_size_area()

A list of shapes can be found at the bottom of the scale_shape documentation.

Exercise 5: Life expectancies in Africa

5(a) Unhiding the data

Origional Code

gapminder %>% 
  filter(continent == "Americas") %>% 
  ggplot(aes(country, lifeExp)) + 
  geom_point() +
  geom_boxplot() +
  scale_x_discrete()

Fixed Code

gapminder %>% 
  filter(continent == "Americas") %>% 
  ggplot(aes(country, lifeExp)) + 
  geom_boxplot() +
  geom_jitter(alpha = 0.2) +
  coord_flip()

5(b) Ridgeplots

Origional Code

gapminder %>% 
  filter(continent == "Americas") %>% 
  ggplot(aes(country, lifeExp)) + 
  geom_point() +
  geom_boxplot()

Fixed Code

gapminder %>% 
    filter(continent == "Americas") %>% 
    ggplot(aes(lifeExp, country)) + 
    ggridges::geom_density_ridges()

Exercise 6: Bar plot madness

6(a) Colour and stacking madness

Origional Code

mtcars %>% 
  mutate(transmission = if_else(am == 0, "automatic", "manual")) %>% 
  ggplot(aes(cyl)) +
  geom_bar(aes(color = transmission))

Fixed Code

mtcars %>% 
  mutate(transmission = if_else(am == 0, "automatic", "manual")) %>% 
  ggplot(aes(cyl, ..prop..)) +
  geom_bar(aes(fill = transmission), position = "dodge") +
  scale_x_continuous(breaks = c(4, 6, 8))
